# Python Refresher Topics

This folder contains the Python refresher materials for the Intel course, designed for a 60–90 minute session with hands-on work.

## Session structure

1. Core Python basics
2. Collections
3. Control flow
4. Functions and behavior
5. Object-oriented programming
6. Data-oriented file I/O and simple aggregation

## Topic list

### 1. Core Python basics
- Variables and assignment
- Primitive types: `int`, `float`, `str`, `bool`
- Type conversion and string formatting
- Boolean logic and truthiness

### 2. Collections
- `list`, `tuple`, `set`, `dict`
- Indexing, slicing, iteration
- Common methods: `append`, `extend`, `pop`, `keys`, `items`, `update`
- Unpacking and multiple assignment

### 3. Control flow
- `if` / `elif` / `else`
- `for` loops and `while` loops
- `break`, `continue`
- `zip`, `enumerate`
- Loop patterns for aggregation and filtering

### 4. Functions and behavior
- Defining functions with `def`
- Positional and keyword arguments
- Default values
- `*args` and `**kwargs`
- `lambda` expressions
- Docstrings and readable function names

### 5. Object-oriented programming
- What is an object?
- Class vs instance
- `__init__` and constructor behavior
- Attributes as stored data
- Methods as actions on the object
- `__repr__` and `__str__`
- Creating and using objects
- Simple composition: objects containing other objects
- Optional extension: inheritance and class/static method basics

### 6. Data-oriented file I/O
- Reading text and CSV files with `open()`
- Parsing lines and splitting values
- Building lists/dicts from rows
- Simple aggregation and summary logic
- Connecting file data to objects


### 7. Errors and exceptions
- `try` / `except` blocks
- `else` and `finally`
- Raising exceptions with `raise`
- Common exception types: `ValueError`, `TypeError`, `KeyError`, `FileNotFoundError`
- Basic use of `with` / context managers for file handling
